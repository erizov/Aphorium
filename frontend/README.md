# Aphorium Frontend

React + Material UI frontend for Aphorium quote search.

## Setup

```bash
cd frontend
npm install
```

## Development

```bash
npm run dev
```

Frontend will run on http://localhost:3000

## Build

```bash
npm run build
```

## Features

- Material UI design
- Responsive layout
- Bilingual search
- Real-time results
- Clean, professional interface
